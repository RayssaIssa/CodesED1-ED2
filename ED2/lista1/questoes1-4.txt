Questão 1 - Defina com as suas palavras o problema de ordenação.

    O problema de ordenação consiste na necessidade de organizar(ordenar) elementos de diversos tipos de dados a fim de reduzir problemas como tempo de busca, inserção e remoção de elementos na computação. Para diferentes problemas e tipos de dados devemos buscar sempre a melhor solução para cada um, então surge a necessidade de criar algoritmos de ordenações cada vez mais poderosos.

Questão 2 - Forneça um exemplo de aplicação real que envolva o problema de ordenação e de encontrar o menor valor.

    Um problema comum de ordenação é em clínicas hospitalares onde se tem um enorme volume de dados a serem acessados todos os dias, é essencial ser capaz de aplicar nesses locais as melhores técnicas de ordenação com a menor percentage de erro e atraso possível.
    Para problemas de encontrar o menor valor temos em vários em relação a trajeto, por exemplo, quando se precisa encontrar a melhor rota de um ponto a outro, com o menor número de tráfego (veículos) possível.

Questão 3 - Qual é a primeira ideia que você consegue pensar para escrever um algoritmo que resolva o problema de ordenação? Faça um programa em linguagem C para demonstrar a sua ideia.

    A primeira coisa que me vem em mente seria a ordenar comparando os valores e os organizando de maneira crescente.
    Código:

    #include<stdio.h>
    int main(){
        int vetor[5] = {5, 9, 3, 10, 1};
        int i, menor = 0;
        for(i = 0; i < 5; i++){
            menor = i;
            for(j = 0; j < 5; j++){
                if(vetor[menor] > vetor[i]){
                    menor = i;
                }
            }
            aux = vetor[i];
            vetor[i] = vetor[menor];
            vetor[menor] = aux;
        }
    }
